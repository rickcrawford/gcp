The catalog also has images that have been pre-computed into 50px x 50px data-uris so they load immediatley. Since this requires a request of the original asset it has been broken off from the regular index process into a separate cloud function. Products are pushed onto an update topic, then asets are fetched/processed and pushed onto another topic for indexing.

The source for this project is available here: https://github.com/rickcrawford/gcp/tree/master/cloudfunctions/resizeImage
