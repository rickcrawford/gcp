Machine learning has advanced to the point where basic models can be built on your machine in minutes. The tokens created by the data flow project can be loaded into a model for further analsys to suggest what the next term should be. Word2Vec is a model that converts ngrams into word vectors for this type of analysis. I have a sample project in python which starts a service that can be used for related terms. This is important in search if you want to do synonym expansion in search.

Here's a couple of projects to learn more:

  • https://medium.com/towards-data-science/learn-word2vec-by-implementing-it-in-tensorflow-45641adaf2ac
