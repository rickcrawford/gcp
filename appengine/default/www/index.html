Autocomplete Examples

The following projects provide you autocomplete functionality of your catalog on product names. There are 3 different implementation options that build off of each other.

  • App Engine Datastore
  • App Engine Flex
  • Elastic Search

If you choose to do further analysis of your catalog there are a wealth of tools available. I put together some additional sample projects for you:

  • Resizing Images using Cloud Functions/PubSub
  • RUM Beacon for BigQuery
  • NGrams with Dataflow
  • Word2Vec related terms
